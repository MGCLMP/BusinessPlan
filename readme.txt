
借入金について

一応町田市で利息等は全額補助を受けることが出来るけど、
念のため計算しています。

借入金はあくまでも予備費用に用いる為概算計算です。
利息分だけ返して行き、必要無くなったら借入金を
全額返済する形です。

売り上げ算定計算方法

売り上げ　＝　ユーザー数　×　単価（980円）

この形態のビジネスは突発的に知名度が上がると
それだけ収益も上がりますが、何時その機会が訪れるか不明かつ
プログラミングコンテストの知名度よりユーザー数の上昇見込みは
少数程度そして無退会を想定して計算しています。
